# GMaps-app
GMaps-app (Angular, Google Maps API, Nodejs, Express, SQLite, Sequelize)

##How to run the app:

###Client:

<pre>
 cd server
 npm install
 ...
 npm run dev 
</pre> 

###Server:
<pre>
 cd client
 npm install
  ...
 npm start
</pre> 

Now you can open the app in your browser by address:
<pre>
http://localhost:8080/
</pre>
